가동시간 탭 상세플로우

2개 섹션

A. 월별 가동시간 테이블

계획운휴일수 입력
가동일수 5개 항목 입력 → 소계 자동계산
비가동일수 3개 항목 입력 → 소계 자동계산
총 조업일수 자동계산 + 분 환산 표시
[저장] → DB 반영

B. 지종별 생산성 마스터

지종 + 평량 + 지폭 + 선속 입력
이론생산성 = 평량 x 지폭 x 선속 x 1440 x 10⁻⁹
폐품율 설정 (기본 1.22%)
양품생산성 = 이론생산성 x (1 - 폐품율)
생산량 계산에 사용: 가동일수 x 양품생산성 = 월 생산량(톤)
